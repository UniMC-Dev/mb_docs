## ✅ Minecraft 白名单（Whitelist）功能说明【管理员专用】

白名单（Whitelist）是一种玩家访问权限控制机制。当服务器启用白名单后，**只有在白名单中的玩家才可以加入服务器**，其余玩家会被拒绝连接。

这项功能广泛用于：

- 私人服务器
- 内测服或施工服
- 防止陌生人进入捣乱
- 临时维护时只允许特定人员登录

---

## ⚙️ 管理白名单

### 🟢 添加白名单：

```bash
/whladd <玩家名>
```

### 🔴 移除白名单：
```bash
/whldel <玩家名>
```

### ⚪ 查看白名单：
```bash
/whllist
```

## ⚠️ 注意事项

- 白名单功能默认关闭，需要管理员权限才能开启。
- 白名单功能仅对登录服务器的玩家有效，不影响已在线的玩家。
- 白名单功能不影响管理员权限。
- 白名单功能不影响服务器内的任何操作。
- 白名单功能不影响服务器的性能。
